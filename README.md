# next-ajax
> Ajax wrapper for next.

## installation
```bash
npm install -S afeiship/next-ajax --registry=https://registry.npm.taobao.org
```

## apis

| fetch | -      | Fetch data. |

```js
import NxAjax from 'next-ajax';

// code goes here:
```
